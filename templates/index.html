About

Hopper is a tool that lets you customize the websites and services used to view AT-URIs.

Form

Use the form to jump to an AT-URI provider for an AT-URI.

Query String Parameters

The following query string parameters are supported:

  • aturi - The URL encoded AT-URI to jump to.
  • server - (Optional) The hostname of an AT-URI provider that serves /.well-known/host-meta.json link templates.

Examples:

  • https://hopper.at/?aturi=at%3A%2F%2Fdid%3Aplc%3Atgudj2fjm77pzkuawquqh...vxj7blk2t
  • web+at://did:plc:tgudj2fjm77pzkuawquqhsxm/community.lexicon.calendar.event/3kxbvxj7blk2t

Pro Tip: Install the Hopper Firefox Extension to open web+at://... URIs
